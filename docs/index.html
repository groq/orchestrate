Quick Start

Get up and running in seconds. Run this command from any git repository:

# Launch agents to fix a bug
orchestrate --name fix-bug --prompt "Fix the login timeout issue"

This creates isolated git worktrees and launches each agent in a separate iTerm2 pane.

Installation

# Clone and build
git clone https://github.com/your-org/orchestrate
cd orchestrate
go build -o orchestrate .

# Move to your PATH
mv orchestrate /usr/local/bin/

Requirements: macOS with iTerm2, Go 1.21+, and your preferred AI coding agents installed.

CLI Reference

Flag Description
--name Required. Branch name prefix for worktrees. Each branch gets a unique hex suffix.
--prompt Required. The prompt to pass to each agent.
--preset Use a preset from .orchestrate.yaml. Defaults to the config's default preset.
--n Multiplier for agent windows. --n 2 runs each agent twice.

Examples

Basic Usage

# Use default preset
orchestrate --name feature-auth --prompt "Add OAuth2 login"

Using Presets

# Use the "heavy" preset
orchestrate --name big-task --preset heavy --prompt "Implement full test coverage"

Scale with Multiplier

# Run each agent in the preset 3 times
orchestrate --name massive --preset mix --n 3 --prompt "Optimize queries"
Each agent runs in its own git branch, so they won't conflict. Compare results and pick the best solution!

Agents must be installed and available in your PATH. Orchestrate passes your prompt directly to them.

Configuration

Create .orchestrate.yaml in your project root to define presets:

# .orchestrate.yaml

# Default preset when --preset is not specified
default: standard

# Presets are lists of windows (agents or custom commands)
presets:
  standard:
    - agent: droid
    - agent: codex
    - agent: codex

  mix:
    - agent: droid
    - agent: claude
    - agent: codex

Custom Commands

Mix agents with custom commands. Order is preserved exactly as defined:

presets:
  dev:
    # Start dev server first
    - command: "npm run dev"
      title: "Dev Server"
      color: "#00ff00"
    
    # Then an AI agent
    - agent: claude
    
    # Then test watcher
    - command: "npm run test:watch"
      title: "Tests"
      color: "#ffff00"
    
    # Another agent
    - agent: codex

How It Works

1. Git Worktrees

Orchestrate creates isolated git worktrees in ../orchestrator-worktrees/. Each agent gets a new branch based on your current branch.

2. iTerm2 Integration

Windows open automatically with up to 6 panes each in a grid layout. Each pane is color-coded by agent type or custom color.

3. Parallel Execution

Agents and commands run in the exact order defined. When agents finish, compare branches and merge the best solution.
